# TNG PayForward
**Borrow for today, build for tomorrow**

## 📌 Overview

TNG PayForward is a financial inclusion feature designed for integration within Touch 'n Go eWallet. It aims to empower underserved users such as low-income individuals, gig workers, and the unbanked by providing access to responsible short-term credit while encouraging savings.

Unlike traditional Buy Now, Pay Later (BNPL) services that may lead to debt cycles, PayForward promotes financial resilience by combining credit access with guided saving behavior.


## 🚀 Problem

Many underserved users face:

* Limited access to formal credit due to lack of banking history
* Irregular income, making it difficult to save consistently
* Existing BNPL solutions that encourage overspending and debt accumulation

This results in a cycle of:
**borrow → repay → borrow**, without long-term financial improvement

## 💡 Solution

TNG PayForward introduces a responsible PayLater system with built-in financial guidance:

* Access short-term credit for essential spending
* Repay in a fixed **4-week plan (0% interest)**
* Receive **Smart Save prompts** after each repayment
* Build an **emergency savings fund over time**


## ⚙️ Key Features

### 🏠 Home Dashboard

* Displays available credit limit
* Shows repayment status and due dates
* Indicates user risk level (Low / Medium / High)
* Provides a quick financial overview

### 💰 Savings Page

* Tracks emergency fund growth
* Displays financial summary (income, expenses, savings)
* AI-powered saving recommendations
* Savings history and progress charts

### 💳 Pay Page

* Enables PayForward transactions
* Shows repayment breakdown and schedule
* Supports seamless weekly repayments


## 🔑 Core Mechanics

* **4-week fixed repayment model**

  * 0% interest within repayment period
  * 2% interest applied only if overdue

* **Risk-based credit scoring**

  * Based on transaction behavior instead of traditional credit scores

* **Smart Save prompts**

  * Users can optionally save small amounts during repayment

* **Emergency Fund Tracker**

  * Encourages long-term financial stability


## 🎯 Impact

TNG PayForward helps:

* Increase access to safe, short-term credit
* Encourage consistent saving habits
* Reduce reliance on high-risk borrowing
* Improve financial resilience for underserved communities


## 📢 Closing

TNG PayForward transforms PayLater from a debt tool into a financial empowerment tool by helping users build savings while managing their spending.
